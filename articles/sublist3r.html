Sublister is a tool designed in python and uses OSINT in order to enumerate subdomains of websites.

 

How to install Sublist3r?

 

The following command can be used to download from GitHub:

  
 $ git clone https://github.com/aboul3la/Sublist3r.git
     

To download the Python modules needed by the tool;

 
On Windows:
 c:\python27\python.exe -m pip install -r requirements.txt
      
On Linux:
 $ sudo pip install -r requirements.txt
      

With the commands "requests, dnspython, argparse" modules are downloaded.

How to use?

The parameters that the tool makes available are as follows:

 

Example Uses

 
Specifying a specific domain name
 $ python sublist3r.py -d example.com
      
Finding existing subdomains on a specific port
 $ python sublist3r.py -d example.com -p 80,443
      
Setting more subdomains using brute-force
 $ python sublist3r.py -b -d example.com
      
Search using only certain search engines
 $ python sublist3r.py -e google,yahoo,virustotal -d example.com requirements.txt
