Dirb

DIRB is a Web content browser. The link you will give it will be scanned on a dictionary-based basis, links, etc. will be displayed. It is a tool that extracts and lists it to you.

Dirb Tool Example Usage

After typing dirb, we can start scanning by specifying the target, provided that we have "http://" or "https://" at the beginning.

Scan Results

When the scanning process starts, it gives information about the start time, the scanned url or ip address, the wordlist file used.

It gives us information about the results of which directories and which pages it can reach.

Which have been successfully achieved (CODE:200)

Which ones have access ban (CODE: 403) gives information about this.

SIZE: section, it gives how many characters the page content consists of. This information will work for you most of the time.

DIRB does a dictionary-based search. In this, it uses the dictionaries that come as default. If we use it as in the example usage on the previous page, 4612 words will use the common.txt file. To make this scan more comprehensive and get more results;

dirb http://site.com /usr/share/dirb/wordlists/big.txt

dirb http://127.0.0.1 /usr/share/dirb/wordlists/big.txt

By guiding a larger wordlist file like the one above (20,458 words), we can enable it to perform a more comprehensive search. To keep the results we need to record the actions taken.

dirb http://example.com -o output.txt

dirb http://127.0.0.1 -o output.txt

After specifying our destination, we specify a file name with the -o parameter as above and want it to be saved in ".txt" format.
