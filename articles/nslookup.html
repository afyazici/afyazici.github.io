nslookup is a network administration command-line tool for querying the DNS to obtain the mapping between domain name and IP address and DNS records.

nslookup, Works via terminal.

nslookup example.com

Name: example.com
Address: 46.20.7.77

To query Mx;

 nslookup -query=mx example.com
     

To query the name server;

 nslookup -query=ns example.com
     

To query SOA record;

 nslookup -type=soa example.com
     

To query all the above values at once;

 nslookup -type=any example.com
