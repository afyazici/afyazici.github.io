Metasploit is an open source exploit framework. Metasploit; It can run in Linux, Windows Mac-OS environments.

 

With Metasploit, it is not just a direct attack. You can make various backdoor files and attack the target system with them.

 

Exploit codes written for vulnerabilities can be found at https://www.exploit-db.com/

   

Metasploit is a framework that includes scanning modules, antivirus circumvention models and already exploits. Kali Linux or Backtrack Linux distributions come with Metasploit installed.

 

The codes to be downloaded should be used after careful examination as they contain backdoor or information leaking codes. Otherwise, you may face serious damage.

   

Exploit: These are the vulnerabilities used to achieve the objectives by using the weaknesses of the services or applications running on the target system.

   

Payload: While accessing the target system, it is loaded into the memory of the target machine, allowing the hacker to perform the desired actions. For example, the code set that allows taking webcam images, obtaining hashdumps, turning on microphone, turning on keyloggers, taking screenshots, locking the keyboard, and so on is called payload. Examples of these payloads are meterpreter, dll injection, passivex.

   

Auxiliary: Additional widgets developed for modules are auxiliary tools. Gathering pre-exploit information is used to advance on the target system after exploitation. One of the most used tools in Metasploit is Aux modules.

 

Encoders: Used to bypass antivirus and security devices. It is used to bypass security devices such as IPS, IDS. It is an additional module that can be used to render a generated malware unrecognizable.

 

Nop :”Not Operation” memory location bits that fill memory for learning purposes. It is often used to deceive intrusion detection and prevention systems.

 

Metasploit can be updated using the "msfupdate" command. We use the "show payload" command to see the payloads, the "Show exploits" to see the auxiliary modules, the "Show auxilary" to view the modules that can be used to bypass the antiviruses, the "Show encoders" command.
