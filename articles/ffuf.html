ffuf is a fest web fuzzer written in Go that allows typical directory discovery, virtual host discovery (without DNS records) and GET and POST parameter fuzzing.

 

Installation

 

You can install ffuf by using one liner command:

  go get github.com/ffuf/ffuf 

or

  git clone https://github.com/ffuf/ffuf  cd ffuf go get go build

To install go,

  apt-get install golang 

Don't forget to add the go/bin directory in PATH variable to use the tool from any directory.

 

To verify proper installation and version of tool,

  ffuf -V 

or

  ~/go/bin/ffuf 

Example Usage

 
Basic command used to brute force website
  
 $ ffuf -w [path-wordlist] -u https://example-url/FUZZ
