Netcat is a networking utility with the help of TCP/IP protocol which reads and writes data across network connections.

 

Basic Usage of Netcat:

  
 $ nc [options] [IP] [port]
    

port: The port number or numbers of the destination. So more than one port can be listened.

Options

 

Basic Netcat Client:

  
 $ nc [Target IP] [port]
     

Here, we initiate a connection on the desired port on the target IP in client mode.

 

Basic Netcat Listener:

  
 $ nc -l -p [Local Port]
     

Here, we create a Netcat Listener on the desired local port in listener mode.
