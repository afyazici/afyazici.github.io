What is SSH

 

SSH and TELNET basically serve the same purpose to connect to a remote server.

 

The difference between SSH and TELNET is that TELNET is an unencrypted protocol and SSH is an encrypted protocol.

   

How does SSH work?

 

One of the devices is designated as the SSH server and the other as the SSH client.

 

During the connection phase, the client must connect to the SSH server and authenticate.

 

In this verification phase, ‘Public Key Encryption’ is used.

 

After approval, the user gains access to the system.

 

SSH Connection Example

 

Let's assume that SSH is used on port 22 and it is open to access.

  
ssh [username] @ [IP]
